Car Workshop Booking System

Node.js Express.js MongoDB REST API HTML/CSS JavaScript
April 2026

A full-stack web application that replaces a legacy PHP/MySQL car workshop booking system with a modern Node.js/Express backend connected to a MongoDB database. The system enables customers to book workshop appointments and gives admins full control over scheduling and capacity.

  • Built a full-stack car workshop appointment booking system with a Node.js/Express backend and MongoDB database, replacing a legacy PHP/MySQL architecture.
  • Designed 5 RESTful API endpoints handling booking creation, retrieval, update, and deletion with server-side validation, duplicate detection, and mechanical capacity enforcement.
  • Developed an admin dashboard with live search, status filtering, and CRUD operations powered by MongoDB aggregation pipelines.

Hostel Management System

PHP MySQL HTML/CSS
Jan – Apr 2025

A multi-role web-based hostel management system built in PHP with separate portals for admin and students. The system is backed by a normalized relational MySQL database.

  • Built a multi-role web-based hostel management system in PHP with separate portals for admin and students, backed by a relational MySQL database.
  • Implemented student-facing features including room booking, leave requests, visitor request submission, maintenance reporting, and personal account management.
  • Developed a full admin dashboard with controls for managing students, approving/rejecting room bookings, leave and visitor requests, tracking attendance, and viewing maintenance reports.
  • Designed a normalized relational database schema covering students, rooms, bookings, attendance, leave, visitor, and maintenance records across multiple SQL tables.

Fitness Tracking App

Flutter Dart Provider fl_chart REST Architecture
Mar – Apr 2026

A cross-platform mobile fitness application built in Flutter with 4 core screens — login/signup, health dashboard, and live workout tracking — following a Provider + MVC architecture.
